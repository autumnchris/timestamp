View the source code for this project.

Example Usages

Get the Current Date

To get JSON data for the current date, go to the path /api/timestamp.

Get Other Dates

Alternatively, you can get data for other dates by adding a parameter of a natural language date or a unix timestamp to the /api/timestamp path. Natural language dates must have YYYY-M-D formatting such as /api/timestamp/2018-10-31. Unix timestamps must be a valid integer defined in milliseconds such as /api/timestamp/1540944000000.

Example Output

A successful output should return a unix timestamp and its corresponding UTC date.

{
  "unix": 1540944000000,
  "utc": "Wed, 31 Oct 2018 00:00:00 GMT"
}
